# 📱 专升本备考APP — HBuilderX 零基础打包完整教程

> 适用：Windows/Mac | 难度：零基础可操作 | 预计耗时：30-60分钟

---

## 一、准备工作

### 1.1 注册HBuilderX账号

1. 打开浏览器访问：**https://dev.dcloud.net.cn/**
2. 点击右上角「注册」，用手机号注册账号
3. 注册完成后登录，进入「开发者中心」
4. 点击「实名认证」→ 按提示完成实名认证（支付宝/微信扫码）
   - **为什么需要认证：** 云打包要求实名账号，否则无法使用

### 1.2 下载安装HBuilderX

1. 访问：**https://www.dcloud.io/hbuilderx.html**
2. 选择「正式版」→ Windows 选择 `.exe` 下载，Mac 选择 `.dmg` 下载
3. 下载完成后：
   - **Windows：** 双击安装包 → 选择安装路径（建议D盘）→ 一直点「下一步」→ 完成
   - **Mac：** 双击dmg → 拖拽到Applications文件夹
4. 启动 HBuilderX
5. 首次启动会提示注册/登录 → 用刚才注册的账号登录

---

## 二、配置项目

### 2.1 打开项目

1. 启动 HBuilderX
2. 点击左侧「文件」→「打开目录」
3. 选择项目文件夹：`app/` 目录
   - 路径示例：`D:\桌面内容不能删\专业课软件设计\app`
4. 左侧项目树应该能看到：
   ```
   app/
   ├── index.html
   ├── manifest.json
   ├── css/style.css
   ├── js/license.js
   ├── js/app.js
   ├── data/mechanics.js
   ├── data/surveying.js
   └── res/...
   ```

### 2.2 配置manifest.json

1. 在左侧项目树中，**右键点击 `manifest.json`** →「用编辑器打开」
2. 确认/修改以下配置：

```json
{
  "id": "com.coursecram.examapp",   // ← 改成你自己的包名（如 com.yourname.app）
  "name": "专升本工程力学测量",       // ← APP显示名称
  "version": { "name": "1.0.0", "code": "100" }
}
```

> **包名规则：** com.你的名字缩写.APP名，例如 `com.zhangsan.kaoyan`
> 包名不能与已有APP重复，否则打包失败

### 2.3 替换图标（重要！）

**方法一：使用图标生成器**
1. 在浏览器打开 `res/图标生成器.html`
2. 点击每个尺寸的「下载」按钮，下载PNG文件
3. 将下载的PNG文件放入 `res/` 文件夹

**方法二：自己设计**
| 文件名 | 尺寸 | 用途 |
|--------|------|------|
| icon_48.png | 48×48 | 低清图标 |
| icon_72.png | 72×72 | 标清图标 |
| icon_96.png | 96×96 | 高清图标 |
| icon_144.png | 144×144 | 超清图标 |

## 三、生成签名文件（首次必须）

> Android APK 必须用数字签名，否则无法安装。此步骤只需做一次。

### 3.1 使用HBuilderX生成签名

1. 在HBuilderX顶部菜单：**「发行」→「原生App-云打包」**
2. 在弹出的打包窗口中：
   - **Android包名：** 填入 manifest.json 中的包名
   - **证书别名：** 填入一个英文名（如 `myapp`）
   - **证书密码：** 设置一个密码（6位以上，**必须记住！**）
   - **证书文件：** 留空，HBuilderX会自动生成

3. 点击「打包」按钮
4. 等待打包完成（首次约5-15分钟）
5. 打包成功后，在项目目录 `unpackage/release/apk/` 找到 `.apk` 文件

### 3.2 保存签名文件（非常重要！）

签名文件位置：HBuilderX 默认保存在用户目录
- Windows：`C:\Users\你的用户名\AppData\Roaming\HBuilder X\`
- Mac：`~/Library/Application Support/HBuilder X/`

**⚠️ 必须备份以下信息，丢失后无法更新APP：**
- 签名文件（.keystore 文件）
- 证书别名
- 证书密码

> 建议将签名文件复制到项目目录 `app/keygen/` 下备份

## 四、云打包操作（每次更新APP都要做）

### 4.1 执行打包

1. HBuilderX → 顶部菜单 **「发行」→「原生App-云打包」**
2. 填写打包信息：

| 配置项 | 填写内容 | 说明 |
|--------|---------|------|
| Android包名 | com.yourname.app | 与 manifest.json 一致 |
| 证书别名 | myapp | 3.1中设置的别名 |
| 证书密码 | ******** | 3.1中设置的密码 |
| 证书文件 | 选择 .keystore | 3.1中生成的签名文件 |
| 打包类型 | 使用云端证书（推荐） | 自动管理签名 |

3. 勾选「**打自定义基座**」— 如果不需要则不勾
4. 点击「**打包**」按钮
5. 等待5-15分钟（取决于排队情况）

### 4.2 查看打包进度

1. 打包开始后，HBuilderX 底部会出现进度条
2. 点击底部的「**控制台**」可以看详细日志
3. 打包完成后会自动打开输出目录

### 4.3 获取APK文件

打包成功后，APK文件在：
```
项目目录/unpackage/release/apk/__UNI__XXXXXXXX_release.apk
```

将APK文件重命名为：`专升本备考_工程力学测量_v1.0.apk`

## 五、安装测试

### 5.1 传输到手机

- **微信传输：** 发送APK到「文件传输助手」→ 手机微信下载
- **USB数据线：** 直接复制到手机Download文件夹
- **QQ传输：** 发送到「我的电脑」

### 5.2 安装

1. 手机打开文件管理器
2. 找到APK文件 → 点击安装
3. **华为/小米/OPPO等手机会提示「未知来源」：**
   - 点击「设置」→ 开启「允许安装未知应用」
   - 返回继续安装
4. 安装完成后，桌面出现APP图标

### 5.3 测试清单

安装后请逐项测试：
- [ ] APP能正常打开（不闪退）
- [ ] 试用期3天倒计时正常显示
- [ ] 输入错误的激活码被拒绝
- [ ] 底部5个阶段导航切换正常
- [ ] 工程力学/工程测量切换正常
- [ ] 知识点卡片展开折叠正常
- [ ] 选择题点击选项有颜色反馈
- [ ] 填空题输入答案能判对错
- [ ] 模拟卷答题完成有分数显示
- [ ] 全屏显示，无地址栏

## 七、常见问题与解决方案

### Q1: 打包时提示"包名已被占用"
**解决：** 修改 `manifest.json` 中的 `id`，改成更独特的包名，如 `com.yourname.lixue2026`

### Q2: 打包时提示"未登录"或"未实名认证"
**解决：** 去 https://dev.dcloud.net.cn/ 登录并完成实名认证

### Q3: 打包后的APK安装后闪退
**解决：**
1. 检查 `manifest.json` 的 `minSdkVersion` 是否 ≤ 手机Android版本
2. 手机连接电脑，用 `adb logcat` 查看崩溃日志
3. 确认手机WebView版本 ≥ 60（2017年后的手机都满足）

### Q4: 打包后显示"解析包错误"
**解决：** 
1. APK可能下载不完整，重新下载
2. 签名文件（.keystore）密码错误，重新设置并打包
3. Android系统版本太低（需 ≥ 5.0）

### Q5: 如何更新APP内容（不需要重新打包）
**解决：** 本题库APP所有内容都在代码中，更新需要：
1. 修改 `data/mechanics.js` 或 `data/surveying.js` 中的题目
2. 修改 `index.html` 中的版本号
3. 重新打包

### Q6: 微信里下载APK后改成了".apk.1"
**解决：** 微信会自动添加后缀，用文件管理器重命名删除 `.1`，或通过QQ发送

### Q7: 打包时提示"SDK版本不匹配"
**解决：** 在 manifest.json 中确认 `targetSdkVersion: "30"`，一般无需修改

### Q8: 华为手机安装时提示"恶意应用"
**解决：** 这是华为安全检测的正常提示（未在华为应用市场上架），点击「继续安装」即可。如果想去除提示，需提交华为应用市场审核

## 八、进阶：自定义品牌

### 8.1 修改APP名称
编辑 `manifest.json`，修改 `"name"` 字段

### 8.2 修改配色
编辑 `css/style.css`，修改 `:root` 中的颜色变量：
- `--primary: #1a73e8;` → 主色调（标题栏、按钮等）
- `--accent: #ff6d00;` → 强调色

### 8.3 修改试用天数
编辑 `js/license.js`，搜索 `totalDays: 3`，将 3 改成你想要的天数

### 8.4 替换题目内容
编辑 `data/mechanics.js` 和 `data/surveying.js`，按现有格式添加/修改题目
